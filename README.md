# Islam AI Tool 🕌

An interactive command-line Islamic knowledge assistant that provides information about the Quran, Hadith, pillars of Islam, daily prayers, duas, prophets, and more.

## Features

- **Five Pillars of Islam** — Shahada, Salah, Zakat, Sawm, Hajj
- **Six Pillars of Faith** — Belief in Allah, Angels, Books, Prophets, Day of Judgment, Divine Decree
- **Quran Verses** — Curated verses on various topics with surah references
- **Hadith Collection** — Authentic sayings of the Prophet Muhammad (peace be upon him)
- **Daily Prayers** — All five prayer times, rakaat counts, and descriptions
- **Common Duas** — Supplications with Arabic text, transliteration, and meaning
- **Prophets** — The 25 prophets mentioned in the Quran
- **Islamic Calendar** — The 12 Hijri months
- **Topic Search** — Ask about patience, mercy, knowledge, gratitude, and more

## Usage

```bash
python3 islam_ai.py
```

Then type your questions naturally:

```
🕌 You: What are the five pillars of Islam?
🕌 You: Tell me about daily prayers
🕌 You: Show me a hadith about kindness
🕌 You: What does the Quran say about patience?
🕌 You: Show me some duas
🕌 You: help
```

Type `quit` or `exit` to leave.

## Project Structure
